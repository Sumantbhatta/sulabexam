Procedures for 3.txt: Install Google App Engine (Python)



Step 1: Install Python 



Download Python: Go to the Python official website and click "Download" for the latest version.



Install Python: Run the installer.



Important: Check the box that says "Add Python 3.x to PATH".

Click "Install Now" and follow the instructions.



Verify Python Installation: Open Command Prompt and type python --version.



Step 2: Install Google Cloud SDK 

(Google App Engine is part of the SDK) 



Download Google Cloud SDK: Visit the Google Cloud SDK page and download the installer for Windows.



Install Google Cloud SDK: Run the installer and follow the on-screen instructions.

Initialize the SDK:

Open Google Cloud SDK Shell (a shortcut should be on your desktop).

Type gcloud init and press Enter.

Follow the instructions to log in to your Google account.

Step 3: Create a "Hello, World!" App in Python 



Create a Project Directory: Create a new folder for your app.



Open Command Prompt in the Directory: Navigate to the folder.

Set Up a Virtual Environment (Optional):

Run python -m venv env.

Activate it by running env\Scripts\activate.

Step 4: Deploy to Google App Engine 



Navigate in Google Cloud SDK Shell: Use the cd command to navigate to your project directory.



Deploy Your Application: Run gcloud app deploy. Follow the prompts.



Open Your Application: After deployment, access your app via the URL provided by GAE.
